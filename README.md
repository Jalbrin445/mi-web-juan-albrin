# Sistema de Autenticación y Registro con Flask y MySQL

Este es un proyecto eductativo para desarrollar dos interfaces de usuario web sencillas con HTML, CSS, Python (Flask) y Jinja2, la cual se tratan del registro de usuario e inicio de sesión.

## Tecnologías Utilizadas.

* **Backend:** Python + Flask
* **Base de Datos:** MySQL + SQLAlchemy 
* **FrontEnd:** HTML, CSS y Jinja2 (Motor de Plantillas Integrado con Flask)
* **Gestión de DB:** MySQL Workbench

## Configuración del entorno.

1. Clonar el repositorio al espacio local.

    En CMD: 
        git clone https://github.com/Jalbrin445/mi-web-juan-albrin.git

2. Pararse en la carpeta (si ya se encuentra entonces omitir este paso):
    cd ../mi-web-juan-albrin
    **NOTA:** La ruta puede varíar según la ubicación donde se haya clonado el proyecto.

3. Crear y activar el entorno virtual.
    En CMD:
        python -m venv venv
        **Windows:** venv\Scripts\activate.bat
        **MAC:** source venv/bin/activate

4. Instalar dependencias:

    En CMD:
        pip install -r requirements.txt


**NOTA:** Proximamente se actualizarán todos las dependencias según necesidades del proyecto.
